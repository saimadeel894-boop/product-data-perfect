# Product Data Architect

B2B product data extraction, validation, and WooCommerce import tool for Transeo Africa.

## Features

- Extract product data from PDFs and web sources
- Validate against Transeo Africa's product standards
- Generate WP All Import compatible JSON
- Direct WooCommerce REST API integration

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase Edge Functions

## Development

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```
